Les technologies du Big Data

Pour gérer le Big Data, on utilise des outils capables de stocker et analyser une grande quantité de données rapidement. Voici les technologies les plus connues.

1) Stockage (où on met les données)

  • Base de données SQL : données bien organisées (ex : MySQL)
  • Base de données NoSQL : données plus variées (ex : MongoDB)
  • Cloud : stockage sur Internet (ex : Google Drive, AWS)

2) Traitement (comment on analyse les données)

  • Hadoop : traite de très grosses données sur plusieurs ordinateurs
  • Spark : plus rapide, utilisé pour l’analyse moderne

3) Visualisation (comment afficher les résultats)

Après l’analyse, il faut comprendre les résultats avec des graphiques.

  • Tableau : outil très utilisé pour faire des dashboards
  • Power BI : outil de Microsoft pour les graphiques et rapports
