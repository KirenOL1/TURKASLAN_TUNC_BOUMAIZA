Description

Ce présent site présente le Big Data, un domaine qui concerne la gestion et l’analyse de très grandes quantités de données. Aujourd’hui, ces données proviennent de nombreuses sources comme les réseaux sociaux, les sites internet, les entreprises ou encore les objets connectés.

L’objectif de ce site est de mieux comprendre ce concept et son utilité dans le monde numérique. Dans la page Définition, nous expliquerons rapidement via une définition qu’est le Big Data. Ensuite, dans la page Exemples, nous présenterons des cas concrets d’utilisation du Big Data dans différents domaines comme la santé, le commerce ou les transports.
