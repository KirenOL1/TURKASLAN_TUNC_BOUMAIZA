Petite définition

Le Big Data désigne l’ensemble des données produites en très grande quantité chaque jour, ainsi que les technologies utilisées pour les stocker, les organiser et les analyser. Ces données peuvent provenir de différentes sources comme les réseaux sociaux, les sites web, les entreprises, les capteurs ou encore les smartphones. Le Big Data se caractérise principalement par un volume très important, une vitesse de traitement rapide et une grande variété de formats (textes, images, vidéos, etc.). Grâce à son analyse, il permet d’obtenir des informations utiles afin d’aider à la prise de décision dans plusieurs domaines comme la santé, le commerce, la finance ou les transports.
