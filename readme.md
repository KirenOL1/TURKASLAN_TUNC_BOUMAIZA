## Description
Ce projet consiste en la création d'un site web pour expliquer le principe des big data et leurs intérêts dans les projections et le machine learning.

## Sujet du Site
Le site se concentre sur l'analyse et la visualisation de données massives (Big Data), permettant d'extraire des tendances significatives et de générer des rapports dynamiques basés sur des flux de données en temps réel ou archivés.
